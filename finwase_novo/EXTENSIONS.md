# Extensões ESSENCIAIS para Flutter

## Instalar estas:
- Dart (dart-code.dart-code)
- Flutter (dart-code.flutter) 
- Android iOS Emulator (diemasmds.vscode-android-ios-emulator)
- Amazon Q Developer (amazonwebservices.amazon-q-vscode)
- GitLens (eamodio.gitlens)
- Error Lens (usernamehw.errorlens)
- Bracket Pair Colorizer 2 (coenraads.bracket-pair-colorizer-2)
- Material Icon Theme (pkief.material-icon-theme)

## Remover estas (desnecessárias para Flutter):
- Codeium
- TabNine  
- Local AI Pilot
- Java Extension Pack (se não usar Java puro)
- C/C++ Extension
- Python Extension
- CMake Tools
- Red Hat extensions

## Comandos para instalar essenciais:
code --install-extension dart-code.dart-code
code --install-extension dart-code.flutter
code --install-extension diemasmds.vscode-android-ios-emulator
code --install-extension amazonwebservices.amazon-q-vscode
code --install-extension eamodio.gitlens
code --install-extension usernamehw.errorlens
code --install-extension pkief.material-icon-theme
